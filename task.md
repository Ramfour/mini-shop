# Задание 1: «Мини-магазин с расчётом заказа»

**Цель:** написать PHP-скрипт, который хранит товары, считает стоимость заказа, НДС, скидки и выводит текстовый чек.

---

## Архитектура проекта

Проект состоит из четырёх файлов:

```
index.php       — точка входа, HTML-обёртка
analytics.php   — все вспомогательные функции
shop.php        — данные и расчёты
receipt.php     — вывод чека
```

Порядок подключения в `shop.php`:

```php
require 'analytics.php';
// ... данные и расчёты ...
require 'receipt.php';
```

В `index.php`:

```php
<?php include 'shop.php'; ?>
```

---

## Файл `analytics.php` — функции

### `formatMoney(int $kopecks): string`

Принимает сумму в копейках, возвращает строку вида `199 руб. 99 коп.`

Использовать:
- `intdiv` для получения рублей
- `%` для получения копеек
- `str_pad` или `sprintf` чтобы `5 коп.` выводилось как `05 коп.` 

```php
echo formatMoney(19999); // 199 руб. 99 коп.
echo formatMoney(100);   // 1 руб. 00 коп.
```

## Файл `shop.php` — данные и расчёты

### Массив товаров

```php
$products = [
    [
        'name'     => 'Кофе',
        'price'    => 19999,
        'count'    => 2,
        'category' => 'drink',
    ],
    // ... минимум 5 товаров
];
```

> Цена хранится в копейках, не во float. Минимум 2 категории: `drink`, `food`.

### Массив категорий

```php
$categories = [
    'drink' => 'Напитки',
    'food'  => 'Еда',
    'other' => 'Другое',
];
```

### Расчёт

Для каждого товара посчитать сумму и НДС. Использовать `foreach`.

```php
$itemSum = $product['price'] * $product['count'];
$itemNds = intval($itemSum * 0.2);
```

Посчитать итоги:
- общую сумму (`$totalSum`)
- общий НДС (`$totalNds`)
- общее количество товаров (`$totalCount`)

### Скидка

Скидка **10%** применяется, если `$totalSum > 300000`.

```php
$hasDiscount = $totalSum > 300000;

if ($hasDiscount) {
    $discountAmount = intval($totalSum * 0.1);
    $totalSum = $totalSum - $discountAmount;
}
```

### Подготовка строк для чека

```php
$title      = 'чек заказа';
$titleUpper = mb_strtoupper($title, 'UTF-8');   // mb_strtoupper
$titleLength = mb_strlen($titleUpper, 'UTF-8'); // mb_strlen
```

## Файл `receipt.php` — вывод чека

Вывести чек в формате:

```
========================================
  ЧЕК ЗАКАЗА
========================================

Товар: КОФЕ (Напитки)
Количество: 2
Сумма: 399 руб. 98 коп.

...

----------------------------------------
Итого:   ...
НДС 20%: ...
Скидка:  да/нет
========================================
Товаров в заказе: N
```

Требования к выводу:
- название товара через `mb_strtoupper`
- категория берётся из `$categories` по ключу
- сумма через `formatMoney`
- конкатенация `.` и интерполяция `""` — использовать оба варианта
- `str_repeat` для разделителей
