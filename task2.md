# Задание 2: «История заказов и аналитика»

**Цель:** расширить проект — добавить историю заказов, фильтрацию, сортировку и простую аналитику.

---

## Архитектура

Файлы остаются те же. Расширяешь:

- `analytics.php` — добавляешь новые функции
- `shop.php` — добавляешь массив `$orders`, добавляешь блок аналитики
- `receipt.php` — добавляешь вывод аналитики после чека

---

## Файл `analytics.php` — новые функции

### `formatDate(string $date): string`

Принимает строку `'2026-07-10'`, возвращает `'10.07.2026'`.

Использовать `explode` для разбивки по `-`, затем склеить в нужном порядке через конкатенацию или интерполяцию.

```php
echo formatDate('2026-07-10'); // 10.07.2026
```

---

### `getOrderSummary(array $order): string`

Возвращает строку вида:

```
Заказ от 10.07.2026: Кофе, Вода — 3 100 руб. 00 коп.
```

Использовать:
- `formatDate` для даты
- `implode` для склейки товаров через `', '`
- `formatMoney` для суммы

---

### `getTotalRevenue(array $orders): int`

Возвращает сумму всех заказов.

Использовать `foreach` для накопления суммы.

---

### `getOrdersWithDiscount(array $orders, int $threshold = 300000): array`

Возвращает только те заказы, где `total > $threshold`.

Использовать `array_filter` со стрелочной функцией `fn`.

---

### `sortOrdersByTotal(array $orders, string $direction = 'asc'): array`

Сортирует заказы по полю `total`.

Использовать `usort` и оператор `<=>`. При `'desc'` — по убыванию, при `'asc'` — по возрастанию.

---

### `getItemsFromOrders(array $orders): array`

Собирает все товары из всех заказов в один плоский массив без дублей.

Использовать:
- `foreach` + `array_merge` для сбора
- `array_unique` для удаления дублей
- `array_values` для переиндексации (после `array_unique` ключи могут быть с дырками)

## Файл `receipt.php` — вывод аналитики

После чека добавить блок:

```
========================================
  АНАЛИТИКА ЗАКАЗОВ
========================================

Общая выручка: X руб. X коп.
Заказов со скидкой: N

Все заказы (по возрастанию суммы):
- Заказ от 01.07.2026: Кофе, Печенье — 349 руб. 97 коп.
- ...

Уникальные товары во всех заказах:
Кофе, Печенье, Чай, Шоколад, Вода
========================================
```

Требования:
- заголовок через `mb_strtoupper`
- список заказов через `sortOrdersByTotal` + `foreach` + `getOrderSummary`
- уникальные товары через `getItemsFromOrders` + `implode`
- количество заказов со скидкой через `count(getOrdersWithDiscount(...))`
